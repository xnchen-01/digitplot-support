# DigitPlot Support Site

Static support and privacy pages for App Store Connect.

## Files

- `index.html`: Support URL for App Store Connect.
- `privacy.html`: Privacy Policy URL for App Store Connect.
- `styles.css`: Local CSS only. No external scripts, fonts, analytics, or tracking.

## Suggested GitHub Pages URLs

After publishing this folder to a public GitHub Pages repository:

- Support URL: `https://<github-username>.github.io/digitplot-support/`
- Privacy Policy URL: `https://<github-username>.github.io/digitplot-support/privacy.html`
